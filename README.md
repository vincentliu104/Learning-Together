## 使用說明

1. 建立學習項目分支，格式為 `<who>/<what>`，例如: `vincent/git`
2. 切換到你的分支，在 `/resources/` 建立你的學習項目
3. 學完要提交，請發 pull request(PR)，reviews 驗收: `view change` > `Approve`
    - Reviewers: `vincentliu104`
    - Assignees: 自己
4. 利用 `README.md` 收集學習資源
