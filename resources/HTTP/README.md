# HTTP

網頁攻城獅的必備基礎協定(protocol)知識

## HTTP status code

status code 系列 | 說明
---------------|---
1xx | 參考資訊 (Informational)
2xx | 成功 (Successful)
3xx | 重新導向 (Redirection)
4xx | 用戶端錯誤 (Client Error)
5xx | 伺服器錯誤 (Server Error)
